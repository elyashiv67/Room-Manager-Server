# Room Manager Server

A Node.js server application for managing rooms.

## Installation

```bash
npm install
```

## Running the Server

### Development mode (with auto-reload):
```bash
npm run dev
```

### Production mode:
```bash
npm start
```

## API Endpoints

- `GET /` - Welcome message
- `GET /api/health` - Health check endpoint

## Technologies

- Node.js
- Express.js
- CORS
- Nodemon (for development)

## Port

The server runs on port 3000 by default. You can change this by setting the `PORT` environment variable.
